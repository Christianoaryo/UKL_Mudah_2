# Program dalam bahasa Java untuk menentukan apakah suatu bilangan yang dimasukkan pengguna adalah bilangan ganjil atau genap.

1.Scanner berfungsi untuk memasukkan input berupa bilangan yang nanti akan di tentukan ganjil genapnya

2.Bilangan tersebut kemudian diperiksa menggunakan operator modulus (%) terhadap angka 2.

3.Jika hasil bilangan % 2 kurang dari atau sama dengan 0, maka bilangan dianggap genap.
  Jika tidak, maka bilangan dianggap ganjil.
